Simple prototype visant à illustrer le concept et peut-être fournir une première solution déployée

```
yarn install
export DATABASE_URL="postgresql://[user[:password]@][netloc][:port][/dbname][?param1=value1&...]"
yarn build
yarn build-dep

### Development
yarn run start-dev

### Production (Heroku)
yarn start
```
